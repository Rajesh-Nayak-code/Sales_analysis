# Sales_analysis
Sales Data Analyzer using NumPy  A Python project that uses Object-Oriented Programming (OOP) and NumPy to perform sales data analysis, including total sales, average sales, minimum and maximum sales, above-average sales detection, percentage contribution analysis, and sorting operations.

## Features

- Calculate total sales
- Find average sales
- Find minimum and maximum sales
- Identify sales values above the average
- Calculate percentage contribution of each sale
- Sort sales data

## Technologies Used

- Python(OOPS)
- NumPy

## Example Usage

obj = Sales([1200, 1500, 900, 2000, 1800, 2200, 1700])

print(obj.total_sales())

print(obj.average_sales())

print(obj.min_max_sales())

print(obj.sales_above_avg())

print(obj.percent_contribution())
